页签 Tabs ​


页签切换

使用场景 ​

  • 选中的页签可以选择下划线或者块显示，宽度可改
  • 可以自定义选中的页签展示效果

基本使用 ​

下划线
块
tab1
tab2
tab3
基本
有两种选中的显示模式
<template>
  <CTabs v-model:activeKey="selectedType" :tab-list="selectTypeList" />
  <CTabs v-model:activeKey="activeKey" :tab-list="tabList" :selected-type="selectedType" />
</template>
<script setup lang="ts">
import { TabList, SelectType } from "@components/tabs/types";
import { ref } from "vue";

const selectedType = ref<SelectType>('line');
const selectTypeList = ref<TabList>([
  { label: "下划线", value: "line" },
  { label: "块", value: "block" },
]);

const activeKey = ref<selectedType>(1)
const tabList = ref<TabList>([
  { label: "tab1", value: 1 },
  { label: "tab2", value: 2 },
  { label: "tab3", value: 3 },
]);
</script>

自定义选中显示的宽度 ​

tab1
tab2
tab3
修改宽度
selected-width修改下划线宽度，位置用align可以改
<template>
  <CTabs :activeKey="1" :tab-list="tabList" :selected-width="20" />
</template>
<script setup lang="ts">
import { TabList } from "@components/tabs/types";
import { ref } from "vue";

const tabList = ref<TabList>([
  { label: "tab1", value: 1 },
  { label: "tab2", value: 2 },
  { label: "tab3", value: 3 },
]);
</script>

自定义选中显示的内容 ​

tab1
tab2
tab3
自定义选中背景
插槽修改选中的展示内容，比如插个指纹图之类的
<template>
  <CTabs :activeKey="1" :tab-list="tabList" >
    <template #sliderContent>
      <div class="dot" />
    </template>
  </CTabs>
</template>
<script setup lang="ts">
import { TabList } from "@components/tabs/src/types";
import { ref } from "vue";

const tabList = ref<TabList>([
  { label: "tab1", value: 1 },
  { label: "tab2", value: 2 },
  { label: "tab3", value: 3 },
]);
</script>
<style lang="less" scoped>
.dot{
  position: absolute;
  left: 20px;
  top: 12px;
  width: 5px;
  height: 5px;
  background-color: red;
  border-radius: 50%;
}
</style>

属性 ​

参数说明类型默认值必传
activeKey(v-model)选中key`numberstring`
tabList页签列表Array<{ label: string; value: any }>true
selectedType选中页签背景显示模式`lineblock`line
transitionDuration展开式动画的durationnumber0.2false
selectedWidth选中页签背景显示宽度'hover' \/ 'click''hover'false
align选中页签背景显示位置`leftrightmiddle`
sliderContent插槽自定义选中页签背景展示内容slotfalse
